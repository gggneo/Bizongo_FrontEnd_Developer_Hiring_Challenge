A responsive web application developed using HTML/CSS/Bootstrap and Javascript/jQuery client side scripting.

 - Functionality to list all games from CSV
 - Pagination enabled results with each page displaying 2000 records.
 - Search feature with auto complete functionality.
 - Local storage for storing small amount of data (in search by name functionality) to retrieve search results quickly
 - Search displays the desired game
 - Sort functionality to sort the games by year.
 - Portfolio section on the page itself to convey the list of functionalities provided.
OPEN THE WEB PAGE ON A SERVER TO ALLOWS RETRIVAL OF DATA FROM CSV FILE.
